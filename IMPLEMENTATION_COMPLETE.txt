⚙️ HOW IT WORKS

User Query
    ↓
CloudGemini System enriches prompt with resources context
    ↓
Sends to Gemini API (free, no token needed)
    ↓
Gemini processes with awareness of all tools/commands/models
    ↓
CloudGemini executes referenced tools/commands
    ↓
Returns complete response with results
