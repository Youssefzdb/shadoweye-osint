⚙️ HOW IT WORKS - THE REACT LOOP

REQUEST
  ↓
THOUGHT (Analyze)
  - What does user want?
  - What approach is best?
  - What tools are needed?
  ↓
ACTION (Execute)
  - Run selected tools
  - Execute commands
  - Handle errors
  ↓
OBSERVATION (Analyze)
  - Do we have the answer?
  - Are there issues?
  - Need more information?
  ↓
DECISION (Iterate or Complete)
  - If incomplete → Go back to ACTION
  - If complete → Go to FINAL ANSWER
  ↓
FINAL ANSWER (Synthesize)
  - Combine all information
  - Format in Claude style
  - Provide clear response
  ↓
RESPONSE (Return)

Total time: 1-6 seconds depending on complexity

API ENDPOINTS

GET /api/claude
  Returns status and capabilities

POST /api/claude
  Request:
    {
      "message": "Your question or task",
      "includeThinking": true,    (optional)
      "includeMetrics": true      (optional)
    }
  
  Response:
    {
      "success": true,
      "message": "Claude's response",
      "thinking": {                (if includeThinking)
        "initialAnalysis": "...",
        "steps": [...],
        "finalReasoning": "...",
        "conclusion": "..."
      },
      "metrics": {                 (if includeMetrics)
        "executionTime": 1234,
        "toolsUsed": [],
        "iterationCount": 1,
        "selfCorrected": false
      }
    }
