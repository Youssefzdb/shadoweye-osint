## 🚨 Troubleshooting

### Problem: Nothing works
**Solution:**
```bash
pnpm dev
# Wait for compilation
# Refresh browser
# Clear cache: Ctrl+Shift+Delete
```

### Problem: No response
**Solution:**
- Check internet connection
- Verify localhost:3000 is open
- Check browser console (F12)
- Refresh page
- Restart dev server

### Problem: Old messages
**Solution:**
- Click "Clear Chat"
- Refresh page
- Check browser history

### Problem: Errors in console
**Solution:**
- Check GEMINI_TEST.md
- Review error messages
- Check network tab
- Verify message format
