💻 API REFERENCE
================================================================================

SEND A MESSAGE:
  POST /api/gemini
  
  Request:
  {
    "message": "Your question here",
    "context": [
      {"role": "user", "content": "..."},
      {"role": "assistant", "content": "..."}
    ]
  }
  
  Response:
  {
    "success": true,
    "message": "Gemini's response...",
    "timestamp": "2024-01-15T10:30:00Z",
    "contextLength": 4
  }

GET CONVERSATION CONTEXT:
  GET /api/gemini/context
  
  Response:
  {
    "success": true,
    "context": [...],
    "length": 4,
    "timestamp": "..."
  }
